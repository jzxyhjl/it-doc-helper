# .doc 文件格式支持可行性分析

## 一、背景

当前系统支持的文件格式：
- PDF (`.pdf`)
- Word (`.docx`) - 使用 `python-docx` 库
- PPT (`.pptx`) - 使用 `python-pptx` 库
- Markdown (`.md`)
- TXT (`.txt`)

**需求**：扩展系统支持 `.doc` 格式（Microsoft Word 97-2003 二进制格式）

## 二、决策结果

**最终决定：不实现 .doc 格式支持**

**原因**：
1. 中文编码稳定性风险较高
2. 需要引入外部系统工具（antiword），增加系统复杂度
3. 维护成本较高
4. 用户可以通过转换为 .docx 格式解决需求

**替代方案**：
- 前端和后端添加友好提示，引导用户转换为 .docx 格式
- 提供明确的错误信息和转换建议
